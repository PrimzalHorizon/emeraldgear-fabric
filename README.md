# Emerald Gear 🟢

## Overview

**Emerald Gear** is my first Minecraft mod.  
It adds emerald-based equipment and introduces special villager interactions when using emerald gear.

---

## Features

- **Emerald Armor, Tools, and Weapons**  
  Adds a full set of emerald gear, including armor pieces, tools, and weapons.

- **Random Villager Discounts**  
  Villagers offer discounted trades when:
  - You are wearing atleast **one piece** of emerald armor  
  - You are holding an **emerald-based item**

---

## Installation

1. Install [Fabric Loader](https://fabricmc.net/)
2. Install [Fabric API](https://modrinth.com/mod/fabric-api/)
3. Place the mod `emeraldgear.jar` and `fabric api.jar` file into your `mods` folder
4. Launch Minecraft using Fabric
5. Enjoy!
